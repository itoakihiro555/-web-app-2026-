# API仕様

## POST /api/tasks
説明：〜〜タスクを登録する

リクエスト：
{ "フィールド名1": "例の値", "フィールド名2": "例の値" }
 {"title": "買い物に行く",
  "dueDate": "2026-05-15",
  "status": "todo"}

レスポンス：
{ "フィールド名1": "例の値", "フィールド名2": "例の値" }
 { "フィールド名1": "例の値", "フィールド名2": "例の値" }
 {"title": "買い物に行く",
  "dueDate": "2026-05-15",
  "status": "todo"}

## GET /api/tasks
説明：タスクの一覧を返す

レスポンス：
[{ "フィールド名": "例の値" }]
[
  {"id": 1,
    "title": "買い物に行く",
    "dueDate": "2026-05-15",
    "status": "todo" },
]
